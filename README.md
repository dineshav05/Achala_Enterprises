Welcome to the official repository for the **Achala Enterprises** landing page. This serves as the digital anchor and corporate umbrella directory for our multi-disciplinary business divisions.

## 🌐 About the Enterprise
Achala Enterprises is a modern business house bridging ancient wisdom with modern technology. This single-page application (SPA) routes visitors to our specific verticals:

1. **Achala Value Education Series:** A modern, bilingual life-skills curriculum and smart-board digital hub rooted in Indian heritage.
2. **Stylemanntra Digital:** Our in-house UI/UX and digital product design studio.
3. **Achala Living / SilkStone:** Ayurvedic lifestyle consulting, sustainable products, and life-aging pause therapy.

## 🛠️ Tech Stack & Architecture
This project is built using a "Lean Startup" approach for maximum speed, zero server maintenance, and seamless hosting:
* **HTML5:** Semantic, accessible markup.
* **Tailwind CSS (via CDN):** For rapid, utility-first UI styling without a build step.
* **Google Fonts (Inter):** Clean, modern typography.
* **FontAwesome:** For lightweight vector icons.
* **Formspree:** Backend-as-a-Service (BaaS) handling the contact form submissions directly to the official email.
